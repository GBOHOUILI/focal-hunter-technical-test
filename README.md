# Focal Hunter Technical Assessment

A full-stack mobile application developed as part of the Focal Hunter technical assessment.

## Tech Stack

### Mobile

- React Native
- Expo
- TypeScript
- React Navigation
- TanStack Query

### Backend

- Node.js
- Express
- TypeScript
- PostgreSQL
- Drizzle ORM
- BullMQ
- Redis

## Features

- Product listing
- Product details
- Subscribe to a product using an email address
- Asynchronous email notification scheduling
- REST API
- PostgreSQL database
- Event-driven architecture
